About Snack Dashboard

What is this all about...

The snack dataset was collected by participating students from LAUSD in 2012. They used mobile phones with the Ohmage software.

The Mobilize project introduces concepts like learning from data, statistics and visualization in a playful and engaging way.

The Dashboard allows for visual exploration of the data without the need for advanced software. It is best viewed in Google Chrome.

Pilot Data

Participating students can view their live responses by loggin in the Ohmage system. There is also a public demo dataset available that was collected during the first pilot study.

During the first pilot study, approximately 2500 survey responses were uploaded. For the public demo dataset, only complete responses (including photo and GPS location) were included, resulting in a dataset of approx 1700 records. Also, up to 1 mile of noise added to each GPS location that was recorded from the students home to ensure students privacy.

The raw dataset can be seen here in CSV format. Please do not reuse this data for other purposes without permission.

The Dashboard

The interactive dashboard was developed to visually explore the data. There is both a live version that participating students can use to view each others data, as well as a public static version with a fixed dataset.

Some of the libraries that are used by the dashboard:

  • Leaflet
  • D3
  • D3-cloud
  • Crossfilter
  • dc.js
  • Twitter Bootstrap
  • jQuery{ui}
